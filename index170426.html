Was der Bot kann

Der Bot liest alle meine Substack-Artikel automatisch über den RSS Feed ein und speichert sie als durchsuchbare Wissensbasis (RAG). Wenn du eine Frage stellst, arbeiten vier spezialisierte Agenten zusammen (CrewAI): Einer sucht die relevanten Stellen, einer analysiert den Makro-Kontext, einer ordnet es für die Schweiz ein, und einer formuliert die finale Antwort. Das Ganze läuft auf Python mit GPT-4o-mini als Sprachmodell und wird über die Telegram Bot API ausgeliefert. So kannst du (oder andere) Fragen zu meinen eigenen Analysen stellen — und bekommst Antworten, die auf meinen echten Texten basieren.

Strategy Code Sample

PineScript v5
atr_daily_strategy.pine
//@version=5
strategy("ATR(5) Daily Strategy with 9:30 Price and Stop Loss", overlay=true)

// Calculate ATR(5) on the daily timeframe
atrDaily = request.security(syminfo.tickerid, "D", ta.atr(5))

// Check if current bar is within trading session (9:30 AM to 4:00 PM)
inSession = not na(time(timeframe.period, "0830-1500"))

// Capture the 9:30 price (first bar of the session)
var float open930 = na
isFirstBarOfSession = inSession and not inSession[1]
if isFirstBarOfSession
    open930 := open

// Plot the 9:30 open price
plot(open930, title="9:30 Open Price", color=color.blue, linewidth=2)

// Long entry: close > 9:30 price + (0.3 * daily ATR)
longCondition = close > (open930 + (0.3 * atrDaily))
if longCondition and inSession
    strategy.entry("Long", strategy.long, qty=2)
    strategy.exit("Long Stop", from_entry="Long", stop=open930)

// Short entry: close < 9:30 price - (0.3 * daily ATR)
shortCondition = close < (open930 - (0.3 * atrDaily))
if shortCondition and inSession
    strategy.entry("Short", strategy.short, qty=1)
    strategy.exit("Short Stop", from_entry="Short", stop=open930)

// Close all positions at 4pm
if hour(time) >= 15
    strategy.close("Long")
    strategy.close("Short")

Warum ich das Projekt nicht weiterführe

⚠️ Plattform-Restriktionen
Viele Broker haben ihre Terms of Service bezüglich automatisierter Execution-Modelle verschärft. Ein langfristiger, stabiler Betrieb war unter diesen regulatorischen Unsicherheiten nicht gewährleistet.
🚀 Shift zu generativer KI
Während dieses Projekt rein regelbasiert war, sehe ich im Bereich Agentic AI deutlich größere Hebelwirkungen für die Analyse komplexer Märkte.

Was ich gelernt habe

Dieses Projekt war meine Einführung in die Welt des quantitativen Tradings. Ich habe gelernt, Handelsstrategien in PineScript zu entwickeln und mathematisch fundierte Markt-Edges zu identifizieren.

Dabei habe ich mich intensiv mit Backtesting-Methoden auseinandergesetzt — inklusive Vermeidung von Overfitting und Monte-Carlo-Simulationen zur Robustheitsprüfung. Die Signal-Weiterleitung erfolgte über TradingView-Alerts und einen selbst gehosteten Server.
